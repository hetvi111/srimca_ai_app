PROJECT OVERVIEW
================================================================================
SRIMCA AI Assistant is a comprehensive Flutter mobile application with a Flask 
backend for Shrimad Rajchandra Institute of Management & Computer Applications (SRIMCA). 
The app provides an AI-powered assistant for students, faculty, visitors, and 
administrators with full college management capabilities.

TECHNOLOGY STACK
================================================================================
FRONTEND:
  - Flutter (Dart) - Cross-platform mobile app
  - Firebase (Auth, Messaging, Cloud Firestore)
  - Material Design 3 with custom Navy Blue theme

BACKEND:
  - Flask (Python) - REST API
  - MongoDB Atlas - Database
  - RAG (Retrieval-Augmented Generation) - AI Chat
  - ChromaDB - Vector database for AI

DEPLOYMENT:
  - Backend: Render.com (Production)
  - Database: MongoDB Atlas Cloud

================================================================================
                           HOW TO RUN
================================================================================

BACKEND:
  cd backend
  pip install -r requirements.txt
  python app.py
  Server runs on http://localhost:5000

FLUTTER APP:
  flutter pub get
  flutter run

BUILD APK:
  flutter build apk --release
