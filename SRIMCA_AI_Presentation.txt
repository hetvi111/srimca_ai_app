PROJECT OVERVIEW
================================================================================
SRIMCA AI Assistant is a comprehensive Flutter mobile application with a Flask 
backend for Shrimad Rajchandra Institute of Management & Computer Applications (SRIMCA). 
The app provides an AI-powered assistant for students, faculty, visitors, and 
administrators with full college management capabilities.

1. INTRODUCTION (2-3 minutes)
   ─────────────────────────────────────────────────────────────────────────
   • Welcome & Project Overview
   
   • What is SRIMCA AI Assistant?
     SRIMCA AI Assistant is an intelligent mobile application designed for 
     Shrimad Rajchandra Institute of Management & Computer Applications (SRIMCA). 
     It serves as a comprehensive digital platform that connects students, faculty, 
     visitors, and administrators with college information through AI-powered 
     conversations and automated management systems.
     
     Key Capabilities:
     → AI-powered chatbot (SAI) for instant college information
     → Complete college management (notices, assignments, materials)
     → Role-based access for Admin, Faculty, Student, and Visitor
     → Push notifications for important updates
     → QR code-based visitor management system
     → Web portal for visitor registration
   
   • Purpose: Bridging students with college information via AI
   • Key Highlight: First AI-powered college management app

                         AI SYSTEM DETAILS
================================================================================

SAI (SRIMCA AI Assistant) - Three-tier Response System:

┌─────────────────────────────────────────────────────────────────────────────┐
│ TIER 1: EXACT MATCH                                                         │
│ Purpose: Handle timetable queries with exact responses                      │
│ File: backend/srimca/exact_match.py                                        │
│ Examples: "timetable", "class timing", "exam schedule"                     │
├─────────────────────────────────────────────────────────────────────────────┤
│ TIER 2: GPT WITH RAG (Retrieval-Augmented Generation)                      │
│ Purpose: Generate context-aware answers using knowledge base                │
│ Files:                                                                      │
│   - backend/srimca/retriever.py  : Search knowledge base                  │
│   - backend/srimca/gpt_answer.py  : GPT integration                         │
│   - backend/srimca/build_db.py    : Build ChromaDB vector store            │
│ Process:                                                                    │
│   1. Retrieve relevant docs from knowledge base                             │
│   2. Send context + question to GPT                                        │
│   3. Return generated response                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ TIER 3: FALLBACK (Keyword Search)                                           │
│ Purpose: Answer queries using keyword matching                              │
│ File: backend/srimca/fallback.py                                            │
│ Process:                                                                    │
│   1. Extract keywords from question                                         │
│   2. Search knowledge base for matching answers                             │
│   3. Return best match or default response                                   │
└─────────────────────────────────────────────────────────────────────────────┘

Knowledge Base Categories:
  - Courses & Departments
  - Fees & Scholarships
  - Placements
  - Hostel & Library
  - Admission Process
  - College Events
  - Contact Information
