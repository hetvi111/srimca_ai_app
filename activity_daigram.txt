================================================================================
                        ACTIVITY DIAGRAM LEGEND
================================================================================

SYMBOLS USED:
-------------
+----------+   -->   Start/End Point (Rounded Rectangle)
    |
    v
+----------+   -->   Process/Action (Rectangle)
    |
    v
+----------+   -->   Decision Point (Diamond)
    |
    v
+----------+   -->   Data Display (Rectangle with wavy bottom)

ARROWS:
-------
    |          -->   Vertical Flow
    v          -->   Downward Flow
    <          -->   Left Flow
    >          -->   Right Flow

COLORS IN IMPLEMENTATION:
-------------------------
- Blue (accentBlue #1E88E5)   --> User Actions
- Navy Blue (#001F3F)         --> Navigation/Headers
- Green                       --> Success Paths
- Red                         --> Error/Alternative Paths
- Orange                      --> API Calls/Data Loading


================================================================================
                         FUNCTIONALITY SUMMARY
================================================================================

1. AUTHENTICATION:
   - Registration: Name, Email, Enrollment, Password, Role
   - Login: Email/Enrollment + Password + Role
   - Logout: Secure session termination

2. VIEW PROFILE:
   - Display: Name, Enrollment Number, Email, Course, Semester
   - Actions: Edit Profile, Security Settings, Preferences

3. SRIMCA AI CHAT:
   - 24x7 AI Assistant (SAI)
   - Academic & College-related queries
   - Instant responses
   - Saves conversation history

4. VIEW NOTICES:
   - Notice Title, Description, Date
   - Priority levels (High/Medium/Low)
   - Pull-to-refresh functionality

5. ANNOUNCEMENT NOTIFICATIONS:
   - Real-time push notifications
   - Types: Exam, Event, Deadline, Update
   - Notification history access

6. VIEW CHAT HISTORY:
   - Previous questions and AI responses
   - Date and time stamps
   - Full conversation reference
